# statusnow

A mobile-first web app MVP that shows live visit status for clinics in Bangalore.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- City landing page with clinic search
- Clinic status page showing current waiting status
- Community reporting system (no login required)
- Client-side state management using localStorage
